### 변경 배경 / 이유
> Dashboard에서 CD101 등 Job의 일간 성공률이 0%로 표시되는 문제 발생. 
> 원인: 1) status_code_service가 빈 리스트를 반환하여 성공/실패 분류 실패
>      2) 시간대 처리 일관성 부족

### 변경 전 → 변경 후
- **Before**: 
  - `status_code_service.get_success_codes()`가 `[]` 반환 → 모든 상태가 'uncollected'로 분류
  - `ui.js`에서 `luxon.DateTime.local()` 사용 (브라우저 시간대 의존)
  - Dashboard 일간 성공률: 0% (잘못된 분류)

- **After**:
  - `status_code_service`가 빈 리스트일 때 기본값 사용 (`['CD901']`, `['CD902', 'CD903']`)
  - `ui.js`에서 `dateUtils.js`의 `getKSTNow()` 사용 (KST 기준)
  - Dashboard 일간 성공률: 정상 표시 (성공/실패/진행중 정확히 분류)
