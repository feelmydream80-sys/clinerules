### 변경 사항 요약
카드 요약 대시보드에 표(리스트)와 콩(물리 애니메이션) 모드를 전환하여 볼 수 있는 기능을 추가했습니다.

### 상세 변경 사항

**1. 표/콩 모드 전환 기능**
- 백틱(`) 키 입력 시 토글 버튼 표시/숨김
- 표 모드: 기존 리스트 형태 유지
- 콩 모드: 물리 애니메이션이 적용된 구슬로 표시

**2. 물리 애니메이션 엔진**
- 중력, 충돌, 마찰 물리 시뮬레이션
- 구슬 크기 동적 계산 (수량에 따라 10px~20px)
- 마지막 구슬 떨어진 후 1.5초 후 전체 동시 정지
- 구슬 간 겹침 자동 해소

**3. 시각적 개선**
- 구슬 2배 크기 증가 (가시성 향상)
- 최소 너비 120px 설정 (일관된 카드 크기)
- 페이드 인/아웃 애니메이션
