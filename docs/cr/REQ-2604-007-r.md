### 변경 사항 요약
> 팝업 관리 기능 신규 도입 - 관리자가 공지사항을 등록하고 사용자가 모든 페이지에서 확인 가능

### 상세 변경 사항

**1. 팝업 관리 기능**
- 팝업 등록/수정/삭제 (관리자 페이지)
- 제목, 내용, 이미지, 링크 URL 설정
- 게시 기간 설정 (시작일 ~ 종료일)
- 팝업 크기 및 배경색 커스터마이징

**2. 실시간 미리보기 UI**
- 좌측: 실시간 미리보기 (입력 즉시 반영)
- 우측: 압축된 입력 폼
- 설정 5개를 한 줄에 표현 (너비/높이/색상/상태/숨김일수)

**3. 사용자 기능**
- 모든 페이지에서 팝업 자동 표시
- 최대 5개까지 다중 팝업 표시 (z-index 관리)
- 1-7일 보지 않기 선택 가능 (LocalStorage 저장)
- 이미지 클릭 시 링크 이동

**4. 영향받는 페이지**
- /admin/mngr_sett (관리자 설정 - 팝업 관리 탭)
- 모든 사용자 페이지 (base.html)
