### 변경 사항 요약
> 팝업 위치 설정: 퍼센트 단위 지원 및 위치별 독립 offset 적용

### 상세 변경 사항
**1. 팝업 위치 퍼센트 지원**
- left, right, top, bottom에 % 단위 설정 가능
- 예: left: '80%', top: '10%'

**2. 위치별 독립 offset 적용**
- 같은 위치(TOP_LEFT끼리, CENTER끼리 등)만 겹침 방지 offset 공유
- 다른 위치는 독립적으로 배치

**3. 백엔드 loc 필드 연동**
- create_popup, update_popup에 loc 필드 추가

**4. 드래그 기능 개선**
- getBoundingClientRect 대신 getComputedStyle 사용
- transform 해제 후 left/top 직접 설정
