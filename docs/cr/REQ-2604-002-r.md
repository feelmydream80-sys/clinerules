### 변경 사항 요약

- 메모가 존재하는 그룹에 관리자 설정에서 지정한 색상을 적용하는 기능 추가
- 성공/실패 텍스트에 상태별 색상 적용

---

### 상세 변경 사항

**1. 메모 존재시 그룹 색상 적용**
- 관리자 설정 > 수집 스케줄 설정에 메모 색상 설정 추가
- 메모가 저장된 그룹의 전체 배경색과 텍스트 색상을 설정한 색상으로 적용
- CSS 우선순위 (!important)를 사용하여 최우선 적용

**2. 성공/실패 텍스트 색상 적용**
- 그룹의 성공/실패 텍스트에 상태코드(CD901, CD902)의 색상 적용
- 상위 그룹과 하위 그룹 모두 적용

**3. 영향받는 페이지**
- 주간 수집 현황 히트맵
- 월간 수집 현황 히트맵
- 관리자 설정 > 수집 스케줄 설정
