# ⛔ 전역 잠금 규칙 (모든 작업에 예외 없이 적용)

> 이 섹션은 Claude가 선의로 범하는 과잉 행동을 막기 위한 규칙이다.

## 규칙 목록

1. **지침 파일 자동 수정 금지**: 사용자가 명시적으로 "지침 수정해줘"라고 요청한 경우에만 `.clinerules/` 파일을 수정한다. 지침이 outdated처럼 보여도 수정하지 않는다.

2. **절차 자동 확장 금지**: 요청되지 않은 절차(영향도 분석, CR 보고서, 릴리즈 노트 등)를 자동으로 실행하지 않는다. 각 절차는 명시된 트리거 조건이 충족될 때만 실행한다.

3. **범위 외 코드 수정 금지**: 요청 범위 외 파일은 명백한 오타·문법 오류라도 수정 전 사용자에게 알린다. → 자세한 등급은 [01.legacy-protection.md](01.legacy-protection.md) 참조

4. **Plan Mode 실행 금지**: Plan mode에서는 읽기만. "수행"이라고 명시적으로 요청할 때까지 대기.
