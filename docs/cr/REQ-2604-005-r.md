### 변경 사항 요약
> 팝업(공지사항) 관리 시스템 신규 구현 - 관리자 설정 및 사용자 표시 기능

### 상세 변경 사항
**1. 팝업 관리 기능**
- 관리자 설정 페이지에 "팝업 관리" 탭 추가
- 팝업 CRUD (생성, 조회, 수정, 삭제)
- 이미지 첨부 및 미리보기 기능
- 팝업 표시 기간 설정 (시작일/종료일)
- 팝업 순서 및 활성화 상태 관리

**2. 사용자 팝업 표시**
- 접속 시 활성화된 팝업 자동 표시
- "하루동안 보지 않기" 체크박스 기능
- 다중 팝업 순차 표시 처리
- 반응형 디자인 (모바일/태블릿/데스크톱)

**3. 데이터베이스**
- TB_POPUP_MST 테이블 신규 생성
- 팝업 메타데이터 저장 (제목, 내용, 이미지, 기간 등)
