### 변경 사항 요약
> 시간 함수 및 차트 색상 일관성 개선

### 상세 변경 사항

**1. 시간 함수 개선**
- get_kst_now()가 datetime 객체를 반환하도록 수정
- DAO/Service 레이어에서 시간 함수 사용 방식 개선
- 템플릿에서 .timestamp() 메서드 정상 호출 가능

**2. 차트 색상/명칭 일관성 개선**
- API 응답에 TB_STS_CD_MST 기준 색상/명칭 정보 포함
- 차트 색상/명칭을 TB_STS_CD_MST 단일 출처에서 조회
- 코드 유지보수성 향상

**3. 범례 표시 개선**
- Chart.js 범례에서 undefined 출력 문제 해결
- generateLabels 콜백 구현으로 안전한 레이블 표시

**4. 영향받는 페이지**
- 대시보드 페이지 (dashboard.html)
- 분석 차트 페이지 (chart_analysis.html)
- 그룹 메모 기능
