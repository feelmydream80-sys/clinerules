### 변경 사항 요약
> CD1000 그룹 명칭 버그 수정 및 5자리 이상 입력 방지

### 상세 변경 사항

**1. CD1000 그룹 명칭 버그 수정**
- card_summary에서 CD1000 그룹(예: CD1001)의 명칭이 solcast(CD100)로 잘못 표시되던 버그 수정
- 4자리(CDxxxx) Job ID 그룹 코드 변환 로직 수정
- comp(CD1000 명칭)이 정상 표시됨

**2. 10만 단위 입력 차단**
- 관리자 설정 > 그룹 추가 시 CD10000(5자리) 이상 입력 불가
- 프론트엔드 실시간 검증 (추가 버튼 비활성화)
- 백엔드 API 검증 추가 (双重 방어)
