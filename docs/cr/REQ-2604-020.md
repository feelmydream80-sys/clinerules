### 변경 요약 (1줄, 50자 이내)
> card_summary 설명 표시 및 레이아웃 개선

### 변경 배경 / 이유
> /card_summary 페이지에서 라디오 버튼 '설명(desc)'을 선택하면 설명 값이 표시되지 않는 버그.原因是 JavaScript에 desc 케이스 처리 로직이 없음. 또한 화면 너비가 1400px로 제한되어 여유 공간이 있음.

### 변경 전 → 변경 후
- Before: 라디오 버튼 '설명' 선택 시 코드 값이 그대로 표시
- After : 라디오 버튼 '설명' 선택 시 cd_desc(설명) 값이 정상 표시

### 영향 범위
> /card_summary 페이지의 모든 Job 카드, 툴팁, 그룹 타이틀
