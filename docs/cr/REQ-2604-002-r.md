### 변경 사항 요약

- 데이터 정의/사용자 관리 페이지의 시간 표시 문제 수정
- 공통 모듈(dateUtils.js)을 통한 UTC → KST 변환 통합

---

### 상세 변경 사항

**1. 시간대 KST 통합 처리**
- 서버측 (Python)에서 UTC datetime을 KST로 변환하여_client에 전송
- 클라이언트측 (JavaScript)에서 "GMT" 감지 후 KST로 변환하는 로직 추가
- 데이터베이스 SQL에서 NOW()를 KST 시간대로 저장하도록 수정

**2. 영향받는 페이지**
- 관리자 설정 > 데이터 정의
- 관리자 설정 > 사용자 관리
- 수집 일정
- 대시보드

### 버그 수정

- 데이터 정의 페이지에서 update_dt 시간이 UTC로 표시되는 문제
- 사용자 관리 페이지에서 created_at 시간이 UTC로 표시되는 문제
- 날짜 정렬 시 시간대 차이로 인한 정렬 오류

---

### 개선 사항

- 모든 시간 관련 처리를 공통 모듈(dateUtils.js)로 통합
- 서버/클라이언트 양측에서의 일관된 시간대 처리
- 시간대 정보 (+09:00) 표시로 UTC/KST 구분 가능
