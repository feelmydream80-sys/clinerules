### 변경 사항 요약

3. 장애 코드 차트 텍스트 색상 정상 적용

---

### 상세 변경 사항

**1. 데이터 정의 > 수정 버튼 동작 변경**
- 기존: 상세데이터를 2개 이상 체크하면 수정 버튼 비활성화
- 변경: 체크박스 개수와 관계없이 선택된 행(selectedRow) 기준으로 수정 버튼 활성화/비활성화

**2. 차트 분석 > 기간별 수집 성공률 범례 UI 개선**
- Chart.js 기본 범례 대신 커스텀 범례 적용
- `max-height: 180px; overflow-y: auto; flex-wrap: wrap` 스타일 적용
- 객체(Job ID)가 많아져도 세로 스크롤로 모든 항목 확인 가능
- 각 범례 항목 클릭 시 해당 데이터셋 show/hide 토글 지원

**3. 차트 분석 > 장애 코드별 비율 텍스트 색상 적용**
- 관리자설정 > 수집스케줄설정 > 상태별 표시 설정의 텍스트 색상(txt_color) 적용
- 기존 하드코딩된 `#fff` 대신 API에서 내려주는 `txt_color` 사용
