### 변경 요약 (1줄, 50자 이내)
> 시간 함수 및 차트 색상 일관성 개선

### 변경 배경 / 이유
> 1. get_kst_now()가 문자열을 반환하여 dashboard 템플릿에서 .timestamp() 호출 시 오류 발생
> 2. 차트 색상/명칭이 TB_STS_CD_MST, TB_MNGR_SETT, JavaScript Map 등 여러 출처에서 조회되어 일관성 없음
> 3. 범례에서 undefined가 출력되는 문제

### 변경 전 → 변경 후
- **Before:** 
  - get_kst_now() → 문자열 반환 → 템플릿 오류
  - 차트 색상: TB_STS_CD_MST, TB_MNGR_SETT, 하드코딩 혼용
  - 범례: undefined 출력
  
- **After:**
  - get_kst_now() → datetime 객체 반환 → 템플릿 정상 작동
  - 차트 색상/명칭: TB_STS_CD_MST 단일 출처 사용
  - 범례: 정상 출력 (generateLabels 콜백 추가)

### 영향 범위
> - 대시보드 페이지 (dashboard.html)
> - 분석 차트 페이지 (chart_analysis.html)
