### 변경 요약 (1줄, 50자 이내)
대시보드 eventLog.js icon map 문법 오류 수정

### 변경 배경 / 이유
대시보드 페이지가 빈 화면으로 표시됨. 파이프라인 로그를 추가하여 진단한 결과, eventLog.js의 `||` 연산자 우측 값이 누락되어 JavaScript 문법 오류가 발생, 전체 스크립트 실행이 중단된 것이 원인.

### 변경 전 → 변경 후
- Before: `||` 우측에 값 없음 → `SyntaxError: Unexpected token ','`
- After : `|| '🔔'` 기본값 추가 → 정상 동작

### 영향 범위
- 대시보드 페이지: 이벤트 로그 아이콘 표시

| 1 | `static/js/modules/dashboard/eventLog.js` | 수정 | defaultIconMap의 || 우측 값 누락 수정 (🔔 fallback 추가) |

**변경 코드 요약:**
```javascript
// Before
'CD901': adminSetting.sr_success_icon_code || ,   // ← SyntaxError
'CD902': adminSetting.cf_fail_icon_code    || ,   // ← SyntaxError
'CD904': adminSetting.sr_success_icon_code ||      // ← SyntaxError

// After
'CD901': adminSetting.sr_success_icon_code || '🔔',
'CD902': adminSetting.cf_fail_icon_code || '🔔',
'CD903': adminSetting.cf_warning_icon_code || '🟠',
'CD904': adminSetting.sr_success_icon_code || '🔔'
```
