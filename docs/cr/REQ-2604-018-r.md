### 변경 사항 요약
> 관리자 설정 API 조회 권한을 모든 인증 사용자에게 개방했습니다.

### 상세 변경 사항
**1. API 조회 권한 개방**
- /api/mngr_sett/status_codes (GET): 관리자 전용 → 모든 인증 사용자
- /api/mngr_sett/schedule_settings (GET): 관리자 전용 → 모든 인증 사용자

**2. 설정 변경은 여전히 관리자 전용**
- POST /status_codes/sync: 관리자만
- POST /status_codes/save: 관리자만
- POST /schedule_settings/save: 관리자만

**3. collection_schedule 페이지 개선**
- 일반 사용자도 카드 색상 정상 표시
- 메모 색상 정상 표시
- 가이드 팝업 정상 표시
