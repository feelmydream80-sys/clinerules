데이터정의 수정버튼 개선, 차트범례 세로스크롤, 장애차트 텍스트색상 적용

---

### 변경 배경 / 이유

1. 데이터 정의에서 상세데이터 2개 이상 선택 시 수정 버튼이 비활성화되는 원인 파악 및 해결
2. 기간별 수집 성공률 차트의 객체정보(범례)가 많아질 때 하단이 잘리는 문제 해결
3. 장애 코드별 비율 차트에서 관리자설정의 텍스트 색상(txt_color)이 적용되지 않는 문제 해결

---

### 변경 전 → 변경 후

**1. 데이터 정의 > 수정 버튼**
- Before: `editBtn.disabled = selectedCount > 1` (체크박스 2개 이상 선택 시 비활성화)
- After: `editBtn.disabled = !selectedRow` (선택된 행 기준으로 활성화)

**2. 차트 분석 > 기간별 수집 성공률 범례**
- Before: Chart.js 기본 범례 사용, 객체 많으면 하단 잘림
- After: 커스텀 범례(`#successRateLegend`), `max-height:180px; overflow-y:auto; flex-wrap:wrap` 적용, 세로 스크롤로 모든 항목 확인 가능

**3. 차트 분석 > 장애 코드별 비율 텍스트 색상**
- Before: `datalabels.color: '#fff'` 하드코딩
- After: `txtColors` 배열 추출 후 각 세그먼트별 `txt_color` 동적 적용

---
