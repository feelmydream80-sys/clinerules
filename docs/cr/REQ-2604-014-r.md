### 상세 변경 사항

**1. 접속 로그 시간대 처리 개선**
- PostgreSQL NOW() 함수 대신 Python get_kst_now() 유틸리티 사용
- KST 시간대 일관성 확보
- 데이터 저장 시 timezone 변환 오류 수정

**2. 통계 조회 쿼리 최적화**
- 중복된 AT TIME ZONE 변환 제거
- 조회 성능 향상
- 코드 가독성 개선

**3. 에러 처리 강화**
- log_menu_access 데코레이터 예외 처리 개선
- 로깅 메시지 상세화
