### 변경 요약 (1줄, 50자 이내)
사용자접속정보 탭 데이터 중복/미출력 문제 및 애니메이션 중복 실행 수정

### 변경 배경 / 이유
- 다른 탭 이동 후 재접속 시 데이터 미출력
- 처음 접속 시 애니메이션이 2번 실행됨
- 탭 전환 시 데이터 refresh 로직 개선 필요

### 변경 전 → 변경 후
- Before: 
  - init()에서 refresh() 호출 + ui.js에서 refresh() 호출 = 2번 실행
  - 탭 이동 후에는 refresh() 미호출로 데이터 없음
  - _doRenderTable()에서 updateMonthHeaders() 중복 호출
- After : 
  - init()에서 refresh() 제거, ui.js에서만 refresh(forceReload=true) 호출
  - _doRenderTable()에서 updateMonthHeaders() 제거
  - 중복 호출 방지 및 데이터 정상 출력

### 영향 범위
- 사용자접속정보 탭 전체
- 탭 전환 시 데이터 로드 로직
