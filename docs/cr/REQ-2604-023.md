### 변경 요약 (1줄, 50자 이내)
> 사용자 접속 정보 히트맵/선차트 파스텔 톤 변경 및 월별 총합 우측 표시

### 변경 배경 / 이유
> 기존 데이터 수량에 따라 색상이 변하는 방식에서 일정한 파스텔 톤으로 통일하고, 월별 총합을 차트 우측에 표시하여 가독성 개선

### 변경 전 → 변경 후
- Before: 
  - 히트맵: 데이터 수량에 따라 색상 변화 (연한 파랑 → 진한 파랑)
  - 선차트: 마지막 주차 값만 표시, 파랑 계열 색상
  - 결과 카운트: "5명" span 요소 존재
- After : 
  - 히트맵: 일정한 파스텔 퍼플 (`#C4A5F8`) 적용
  - 선차트: 월별 총합을 차트 우측에 표시, 색상 유지
  - 결과 카운트: span 요소 제거

### 영향 범위
> 관리자 설정 > 사용자 접속 정보 페이지의 테이블 차트 영역
