### 변경 요약 (1줄, 50자 이내)
> 카드 요약 페이지 UI 개선 및 그룹 색상 기능 추가

### 변경 배경 / 이유
> 기존 카드 요약 페이지는 고정된 레이아웃과 단조로운 색상으로 사용자 경험이 떨어졌음.
> 데이터 수집 일정 페이지와 동일한 UX 패턴을 적용하여 일관성을 높이고,
> 성공률 기반 그룹 색상으로 한눈에 상태 파악이 가능하도록 개선.

### 변경 전 → 변경 후
- Before: 
  - 그룹 카드가 세로로만 배치됨
  - 고정된 파란색 그라데이션 헤더
  - Job pill은 단색(흰색)으로 표시
  - 검색창 위치가 라디오 버튼과 분리되어 있음
  
- After :
  - 그룹 카드 flex-wrap 레이아웃 (1~3개 동적 배치)
  - 성공률 기준 CD901/CD902 색상 적용 (투명도 50%)
  - Job pill에 상태코드별 색상 적용
  - 검색창-라디오 버튼 한 줄 배치
  - 상세 데이터 툴팁 기능 추가
