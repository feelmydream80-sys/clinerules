### 변경 사항 요약
시간 처리 로직을 공통함수로 일원화하고, DB 조회 시 문자열 형태로 반환하도록 변경하여 브라우저 시간대 설정과 관계없이 일관된 시간 표시 가능

### 상세 변경 사항
**1. 시간 처리 공통함수 개선**
- getKSTNow() 함수 수정: 브라우저 시간대 독립적 동작
- formatDBDateTime() 함수 추가: DB 시간 문자열 파싱

**2. DAO SQL 쿼리 수정**
- TO_CHAR(created_at, 'YYYY-MM-DD HH24:MI:SS') 적용
- datetime 객체 반환 → 문자열 반환

**3. 데이터 처리 규칙 문서화**
- data-handling-rules.md 신규 생성
- 시간/날짜 처리 표준 규칙 정의
