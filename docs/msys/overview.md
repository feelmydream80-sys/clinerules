# 프로젝트 개요

> MSYS 프로젝트의 기본 정보

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **프레임워크** | Flask + jQuery |
| **데이터베이스** | PostgreSQL (연결 풀 관리) |
| **역할** | 데이터 수집 관리 및 모니터링 웹앱 |
| **타임존** | KST (Asia/Seoul, UTC+9) |

---

## 주요 기능

1. **데이터 수집 관리** - Job별 수집 스케줄 및 이력 관리
2. **시각화** - Jandi 히트맵, 차트 분석
3. **데이터 명세** - 컬럼 매핑, 데이터 스펙 관리
4. **권한 관리** - 사용자별 메뉴 접근 제어
