### 변경 사항 요약
> 수집 일정 페이지에서 발생하는 두 가지 버그를 수정했습니다.

### 상세 변경 사항
**1. 403 Forbidden 에러 해결**
- 게스트(비관리자) 사용자가 /collection_schedule 페이지 접근 시 관리자 전용 API 호출로 인한 403 에러 해결
- isAdminUser 권한 체크 추가하여 관리자만 API 호출하도록 수정

**2. TypeError: Cannot read properties of undefined 버그 해결**
- getStatusInfoByCd('CD901') 호출 시 undefined 접근으로 인한 TypeError 해결
- settingsManager._statusMapByCd: {} 초기화 추가

**3. 메모 버튼 표시 로직 개선**
- 관리자: 메모 없으면 '+', 있으면 '✓' 표시
- 게스트: 메모 없으면 숨김, 있으면 '✓' 표시 (조회만 가능)
