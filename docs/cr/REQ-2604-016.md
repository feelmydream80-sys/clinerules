### 변경 요약 (1줄, 50자 이내)
> card_summary 라디오 버튼 표시 모드 버그 수정

### 변경 배경 / 이유
> /card_summary 페이지에서 두 가지 버그가 발견됨:
> 1. 라디오 버튼(명칭/코드/명칭+코드) 선택 시 그룹카드 타이틀이 변경되지 않음
> 2. 콩 모드에서 구슬 툴팁이 코드값만 표시되고 라디오 버튼 선택값이 반영되지 않음

### 변경 전 → 변경 후
- Before: 그룹 타이틀은 항상 코드값(CD100)만 표시, 콩 모드 툴팁은 원시 job ID 표시
- After : 라디오 버튼 선택에 따라 그룹 타이틀과 툴팁이 동적으로 변경됨

### 영향 범위
> /card_summary 페이지 UI만 해당
