# LLM 데이터 분석 서버 구축 따라하기 (PandasAI + Llama.cpp)

이 문서는 데이터 분석 특화 LLM 서버를 별도 프로젝트로 구축하는 방법을 단계별로 안내합니다.

---

## 1. 사전 준비
- Python 3.8 이상
- pip, 가상환경(venv) 사용 권장
- (옵션) GPU 환경이면 더 빠름, CPU도 가능

```
source llm_env/bin/activate
```

## 4. Llama 모델 다운로드
- [HuggingFace Llama2 GGUF 모델](https://huggingface.co/TheBloke/Llama-2-7B-Chat-GGUF) 등에서
  예: llama-2-7b-chat.Q4_K_M.gguf 파일을 다운로드
- 프로젝트 폴더에 저장

---

## 5. Flask + PandasAI 분석 서버 코드 작성 (예시: server.py)
```
from flask import Flask, request, jsonify
from pandasai import PandasAI
from pandasai.llm import LlamaCpp
import pandas as pd

llm = LlamaCpp(model_path="llama-2-7b-chat.Q4_K_M.gguf")
pandas_ai = PandasAI(llm)

app = Flask(__name__)

@app.route('/analyze', methods=['POST'])
def analyze():
    file = request.files['file']
    prompt = request.form['prompt']
    df = pd.read_csv(file)
    # --- 시간(GMT→KST) 변환 예시 ---
    # 예: 'start_dt', 'end_dt' 컬럼이 GMT(UTC)라면 +9시간(Asia/Seoul)로 변환
    for col in ['start_dt', 'end_dt', 'date', 'log_dt']:
        if col in df.columns:
            df[col] = pd.to_datetime(df[col], utc=True, errors='coerce').dt.tz_convert('Asia/Seoul').dt.strftime('%Y-%m-%d %H:%M:%S')
    # --- 대용량 데이터 실무 분석 예시 ---
    summary = df.describe().to_string()
    sample = df.head(1000)
    llm_prompt = f"아래는 데이터 요약과 샘플입니다.\n[요약]\n{summary}\n[샘플]\n{sample.to_string(index=False)}\n{prompt}"
    result = pandas_ai.run(sample, prompt=llm_prompt)
    return jsonify({'result': result})

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8000)
```

---

## 6. 서버 실행
```

```
